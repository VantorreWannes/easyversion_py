# EasyVersion

Minimal, fast snapshots for folders. Take a snapshot, list snapshots, and materialize any snapshot into a new folder.

--------------------------------------------------------------------------------

## Start here

- Users: how to download and use the app — see [USERS.md](docs/USERS.md)
- Developers: build, local install, tests — see [DEVELOPERS.md](docs/DEVELOPERS.md)

--------------------------------------------------------------------------------

## What it does

- Save: capture the current state of a folder
- List: see saved snapshots
- Split: create a new folder that matches a chosen snapshot
- Clean: remove EasyVersion metadata for the current folder

Safety and data locations, CLI flags, uninstall, and troubleshooting are documented in [USERS.md](docs/USERS.md).
